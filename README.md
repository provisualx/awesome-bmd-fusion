# Awesome BlackmagicDesign Fusion

A curated list of awesome BlackmagicDesign (BMD) Fusion libraries, resources, API snippets

BMD Fusion, former eyeon Fusion and Digial Fusion is a profressional digital compositing software available from [BMD Website](https://www.blackmagicdesign.com).
